
# 데이콘 AI 경진대회 대시보드 - 제품 요구사항 정의서 (PRD)

**버전:** 2.2
**작성일:** 2024-08-05

---

## 1. 개요 및 목표 (Overview & Objective)

### 1.1. 제품 소개
데이콘 AI 경진대회 대시보드는 데이콘(Dacon) 플랫폼에서 제공하는 모든 AI 경진대회, 학습 강좌, 기초 코드 등의 콘텐츠를 통합적으로 탐색하고 관리할 수 있는 원스톱 서비스입니다. 사용자는 강력한 검색, 필터링, 정렬 기능을 통해 자신에게 필요한 정보를 신속하게 발견하고, 'AI 탐험가 로드맵'을 통해 개인화된 학습 경로를 추천받을 수 있습니다.

### 1.2. 문제 정의 (Problem)
- 데이콘에는 수많은 대회와 학습 콘텐츠가 흩어져 있어 사용자가 원하는 정보를 찾기 어렵습니다.
- 초보자는 어디서부터 학습을 시작해야 할지 막막함을 느낍니다.
- 사용자는 자신의 목표(대회 수상, 취업, 스킬업 등)에 맞는 콘텐츠를 체계적으로 추천받고 싶어 합니다.
- 단순 키워드 검색만으로는 연관성이 높은 콘텐츠를 발견하기 어렵습니다.

### 1.3. 목표 (Goals)
- **정보 접근성 향상:** 모든 데이콘 콘텐츠를 단일 인터페이스에서 통합 검색 및 필터링하여 사용자 편의성을 극대화합니다.
- **개인화된 학습 경험 제공:** 'AI 탐험가 로드맵' 기능을 통해 사용자 페르소나(직장인, 입문자, 대회 도전자 등)에 맞는 맞춤형 학습 경로를 제시합니다.
- **지능형 검색 구현:** Gemini AI를 활용한 시맨틱 검색으로 사용자의 검색 의도를 파악하고 연관 콘텐츠를 추천합니다.
- **데이터 안정성 확보:** Dacon API 서비스 레이어를 구축하여 외부 데이터 연동의 안정성과 유지보수성을 확보합니다.

## 2. 핵심 기능 (Core Features)

### 2.1. 통합 콘텐츠 뷰
- **대회 목록 (Competition View):** 데이콘의 모든 경진대회(진행중, 종료, 연습)를 카드 형태로 제공합니다.
- **학습 강좌 (DaySchool View):** 데이스쿨의 강좌, 해커톤, 랭커 특강 콘텐츠를 통합하여 제공합니다.
- **기초 코드 (BaseCode View):** 대회별 베이스라인 코드를 카테고리별로 분류하여 제공합니다.
- **AI 탐험가 로드맵 (Roadmap View):** 페르소나별 맞춤형 학습 경로를 제공합니다.

### 2.2. Dacon API 통합 서비스 (`daconApi`)
- **중앙 집중식 데이터 페칭:** 모든 외부 API 호출을 `services/daconApi.ts`에서 관리합니다.
- **주요 엔드포인트:**
    - `getCompetitions`: 전체 대회 목록 및 상세 정보 수집
    - `getEducationList`: 데이스쿨 프로젝트, 해커톤, 랭커 영상 데이터 통합 수집
    - `getCompetitionStats`: 총상금, 참가자 수 등 실시간 통계 수집
    - `getEducationStats`: 레슨 수, 학습 콘텐츠 통계 수집

### 2.3. 지능형 검색 및 필터링
- **AI 확장 검색:** Gemini AI 기반 연관 검색어 자동 추천.
- **통합 검색바:** 검색어 입력 시 대회, 강좌, 코드를 실시간으로 필터링하며 검색 정렬 시 데이터 무결성을 보장합니다.

## 3. 기술 스택 (Technical Stack)

- **프론트엔드:** React 19, TypeScript
- **스타일링:** TailwindCSS, Custom CSS (Glassmorphism, Neumorphism 스타일)
- **AI 모델:** Google Gemini 2.5 Flash
- **API 레이어:** 전용 서비스 모듈 (`daconApi.ts`)을 통한 데이터 추상화
- **데이터 소스:** Dacon Public API v1/v2 (실패 시 정적 데이터 Fallback)
