# 데이콘 AI 경진대회 대시보드 - 제품 요구사항 정의서 (PRD)

**버전:** 2.1
**작성일:** 2024-08-01

---

## 1. 개요 및 목표 (Overview & Objective)

### 1.1. 제품 소개
데이콘 AI 경진대회 대시보드는 데이콘(Dacon) 플랫폼에서 제공하는 모든 AI 경진대회, 학습 강좌, 기초 코드 등의 콘텐츠를 통합적으로 탐색하고 관리할 수 있는 원스톱 서비스입니다. 사용자는 강력한 검색, 필터링, 정렬 기능을 통해 자신에게 필요한 정보를 신속하게 발견하고, 'AI 탐험가 로드맵'을 통해 개인화된 학습 경로를 추천받을 수 있습니다.

### 1.2. 문제 정의 (Problem)
- 데이콘에는 수많은 대회와 학습 콘텐츠가 흩어져 있어 사용자가 원하는 정보를 찾기 어렵습니다.
- 초보자는 어디서부터 학습을 시작해야 할지 막막함을 느낍니다.
- 사용자는 자신의 목표(대회 수상, 취업, 스킬업 등)에 맞는 콘텐츠를 체계적으로 추천받고 싶어 합니다.
- 단순 키워드 검색만으로는 연관성이 높은 콘텐츠를 발견하기 어렵습니다.

### 1.3. 목표 (Goals)
- **정보 접근성 향상:** 모든 데이콘 콘텐츠를 단일 인터페이스에서 통합 검색 및 필터링하여 사용자 편의성을 극대화합니다.
- **개인화된 학습 경험 제공:** 'AI 탐험가 로드맵' 기능을 통해 사용자 페르소나(직장인, 입문자, 대회 도전자 등)에 맞는 맞춤형 학습 경로를 제시합니다.
- **지능형 검색 구현:** Gemini AI를 활용한 시맨틱 검색으로 사용자의 검색 의도를 파악하고 연관 콘텐츠를 추천합니다.
- **사용자 참여 증진:** 다채로운 테마, 동적인 UI, AI 기반 추천 카드 등을 통해 사용자의 흥미를 유발하고 서비스 재방문을 유도합니다.

### 1.4. 타겟 사용자 (Target Audience)
- **결과로 증명하는 직장인:** 핵심만 빠르게 배워 실무에 바로 적용하고 싶은 분.
- **AI와 이제 막 친해지고 싶은 비전공자:** 코딩과 AI가 처음이라 기초부터 탄탄히 다지고 싶은 분.
- **최신 기술을 섭렵하는 열정가:** 최신 AI 기술 트렌드를 놓치지 않고 다양한 도메인을 정복하고 싶은 분.
- **AI 경진대회 도전자:** 대회 우승과 상위 랭킹을 목표로 실전 모델링 기술에 집중하고 싶은 분.
- **AI 연구원 지망생:** AI 기술의 깊이 있는 이해를 바탕으로 대학원 진학이나 R&D 직무를 꿈꾸는 분.
- **AI 서비스 개발자:** AI 모델을 활용하여 세상에 없던 새로운 서비스나 앱을 만들고 싶은 분.
- **데이터 스토리텔러:** 데이터 속 숨겨진 이야기를 찾아내 비즈니스 가치를 창출하고 싶은 분.

## 2. 핵심 기능 (Core Features)

### 2.1. 통합 콘텐츠 뷰
- **대회 목록 (Competition View):** 데이콘의 모든 경진대회(진행중, 종료, 연습)를 카드 형태로 제공합니다.
- **학습 강좌 (DaySchool View):** 데이스쿨의 강좌, 해커톤, 랭커 특강 콘텐츠를 통합하여 제공합니다.
- **기초 코드 (BaseCode View):** 대회별 베이스라인 코드를 카테고리별로 분류하여 제공합니다.
- **AI 탐험가 로드맵 (Roadmap View):** 사용자 페르소나를 선택하면 목표에 맞는 단계별 학습 로드맵과 프로젝트 퀘스트를 제시하여 체계적인 학습 경로를 제공합니다.

### 2.2. 강력한 검색 및 필터링
- **통합 키워드 검색:** 단일 검색창으로 대회, 강좌, 코드를 한 번에 검색합니다.
- **AI 확장 검색 (Semantic Search):** Gemini AI가 검색어를 분석하여 관련 동의어, 기술 용어(한/영)를 자동으로 추가하여 검색 결과를 확장합니다.
- **검색어 제안:** 검색창 포커스 시 '최근 검색어'와 '인기 키워드'를 추천합니다.
- **상세 필터 (대회):**
    - **상태:** 전체, 진행중, 종료, 연습
    - **유형:** 알고리즘, 프롬프트, 개발, 아이디어
    - **기간:** 시작일과 종료일을 지정하여 기간별 필터링
    - **데이터 다운로드 가능 대회:** 데이터가 제공되는 대회만 필터링
- **상세 필터 (학습):**
    - **종류:** 전체, 강좌, 해커톤, 랭커 특강
    - **인기 키워드:** 추천 키워드 버튼으로 필터링
    - **난이도:** 초급, 중급, 고급
- **상세 필터 (코드):**
    - **카테고리:** 생성AI, NLP, 정형, 비전, 데이터분석, 전처리

### 2.3. 정렬 기능
- **대회 정렬:** 최신순, 마감 임박순, 참가자 많은 순, 상금순
- **학습 정렬:** 신규순, 최신순, 제목순, 난이도순, 학습 시간순, 참여 인원순

### 2.4. AI 기반 추천 및 정보 제공
- **동적 마케팅 배너:** 접속 시마다 Gemini AI가 새로운 학습 관련 마케팅 문구를 생성하여 헤더 배너에 표시합니다.
- **예정 대회 카드 (UpcomingCompetitionCard):** 대회 목록 뷰 첫 페이지에 고정으로 표시되어, 곧 열릴 주요 대회를 미리 알려주어 사용자가 중요한 기회를 놓치지 않도록 돕습니다.
- **AI 필수 지식 카드 (WittyQuoteCard):** 예정 대회 카드 옆에 위치하며, AI가 생성한 데이터 사이언티스트를 위한 필수 지식 및 꿀팁을 제공합니다. '새로고침' 버튼으로 새로운 팁을 얻을 수 있습니다.
- **AI 맞춤 강좌 추천 카드 (DaySchoolAiCard):** 학습 강좌 뷰 첫 페이지에 사용자의 학습 목표에 맞는 강좌를 AI가 추천하고 그 이유를 설명합니다.
- **실시간 데이터 티커:** 데이콘의 총 대회 수, 진행 중인 대회 수, 총상금, 총 참가자 수 등의 통계 정보를 헤더 하단에 실시간으로 표시합니다.

### 2.5. 사용자 경험 (UX/UI)
- **멀티 테마:** 사용자의 취향에 따라 '글래스모피즘', '뉴로모피즘', '웹툰' 3가지 UI 테마를 선택할 수 있습니다.
- **반응형 디자인:** 데스크톱, 태블릿, 모바일 등 모든 디바이스에서 최적화된 화면을 제공합니다.
- **동적 UI/애니메이션:** 카드 로딩 시 스태거 애니메이션, 주요 버튼의 호버/클릭 효과, 토스트 메시지 등을 통해 생동감 있는 사용자 경험을 제공합니다.
- **상태 표시:** 로딩, 오류, 검색 결과 없음 등의 상태를 명확하게 시각적으로 안내합니다.
- **사용자 가이드:** '매뉴얼' 모달을 통해 사이트의 모든 기능을 상세히 설명합니다.

## 3. 기술 스택 (Technical Stack)

- **프론트엔드:** React, TypeScript
- **스타일링:** TailwindCSS, Custom CSS (애니메이션, 테마)
- **빌드 도구:** esbuild
- **AI 모델:** Google Gemini API
- **데이터 소스:** Dacon Public API (실패 시 정적 JSON 데이터로 Fallback)
- **서버:** Express.js (정적 파일 서빙)

## 4. 성공 지표 (Success Metrics)

- **사용자 참여도 (Engagement):**
    - 일일/주간 활성 사용자 수 (DAU/WAU)
    - 평균 세션 시간
- **기능 채택률 (Feature Adoption):**
    - 테마, 뷰 모드, 각종 필터 및 정렬 기능 사용률
    - 'AI 탐험가 로드맵' 기능 사용률 및 페르소나 선택 분포
    - AI 추천 카드(지식, 강좌)와의 상호작용률 (새로고침 클릭 수)
- **핵심 행동 전환율 (Conversion):**
    - 대시보드 내 대회/강좌 카드 클릭 후, 실제 dacon.io 페이지로 이동하는 비율 (Click-Through Rate)
    - 검색 기능 사용률 및 검색 성공률
