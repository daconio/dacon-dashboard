
# 데이콘 AI 경진대회 대시보드 - 제품 요구사항 정의서 (PRD)

**버전:** 2.3
**작성일:** 2024-08-05

---

## 1. 개요 및 목표 (Overview & Objective)

### 1.1. 제품 소개
데이콘 AI 경진대회 대시보드는 데이콘(Dacon) 플랫폼에서 제공하는 모든 AI 경진대회, 학습 강좌, 기초 코드 등의 콘텐츠를 통합적으로 탐색하고 관리할 수 있는 원스톱 서비스입니다. 사용자는 좌측 사이드바와 상단 검색/필터 영역을 통해 자신에게 필요한 정보를 신속하게 발견하고, 'AI 탐험가 로드맵'을 통해 개인화된 학습 경로를 추천받을 수 있습니다.

### 1.2. 주요 UI 업데이트 (Sidebar Architecture)
- **공간 효율성 최적화:** 메인 콘텐츠 영역의 수직 공간을 확보하기 위해 주요 네비게이션을 좌측 사이드바로 이동했습니다.
- **콘텐츠 그룹화:** '대회(Competitions)'와 '학습(Learning)'이라는 두 개의 대분류로 메뉴를 구조화하여 사용자의 인지 부하를 줄였습니다.
- **반응형 대시보드:** 데스크톱 최적화 사이드바를 제공하며, 모바일에서는 유연한 레이아웃 전환을 지원합니다.

---

## 2. 핵심 기능 (Core Features)

### 2.1. 사이드바 네비게이션 (Sidebar Nav)
- **대회 그룹:**
    - `대회`: 전체 경진대회 목록
    - `데이터`: 데이터셋 다운로드 전용 대회
    - `코드`: 기술 카테고리별 베이스라인 코드
    - `참가 방법`: 대회 프로세스 가이드
- **학습 그룹:**
    - `학습`: 전체 교육 콘텐츠
    - `강좌/해커톤/특강`: 종류별 세부 필터링
    - `로드맵`: 페르소나별 학습 경로

### 2.2. Dacon API 통합 서비스 (`daconApi`)
- (기존 내용 유지: 중앙 집중식 데이터 페칭 모듈 운영)

### 2.3. 지능형 검색 및 상세 필터
- **통합 검색바:** 검색어 입력 시 전 영역(대회, 강좌, 코드) 동시 필터링.
- **동적 필터 버튼:** 현재 선택된 메뉴(사이드바)에 따라 최적화된 상세 필터 옵션 제공.

---

## 3. 기술 스택 (Technical Stack)

- **프론트엔드:** React 19, TypeScript
- **스타일링:** TailwindCSS, Custom CSS (Themes: Glass, Neumorphic, Webtoon)
- **AI 모델:** Google Gemini 3 Flash Preview
- **레이아웃 구조:** Sidebar + Main Content Flex Layout

---

## 4. 성공 지표 (Success Metrics)

- **네비게이션 효율성:** 메뉴 전환 시 발생하는 지연 시간 및 사용자 클릭 횟수 감소
- **데이터 연동 성공률:** API 호출 성공 및 데이터 정합성 유지 비율
- **사용자 경험 점수:** 테마 기능 및 사이드바 레이아웃에 대한 긍정적 피드백
